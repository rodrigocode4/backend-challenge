# Desafio de código Back-end


## Rodando o projeto base

Para rodar o projeto, execute em um terminal o comando:

```
./gradlew bootRun
```
## Rotas Principais
```
/api/trips/
/api/trips/{VehicleId}
/api/trips/{plate}
```
----
